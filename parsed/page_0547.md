## 928 Covariance Review VI

Company : None

## Prompt:

Suppose that Z ∼ N (0 , 1) and Y ∼ χ 2 ( ν ) are independent. Define W = Z/ √ Y . Compute Cov ( Y, W ) .

## Explanation:

We proceed as follows, making use of the fact that Y and Z are independent:

$$C o v ( Y , W ) & = \mathbb { E } [ Y W ] - \mathbb { E } [ Y ] \mathbb { E } [ W ] \\ & = \mathbb { E } [ Y W ] - \mathbb { E } [ Y ] \mathbb { E } \left [ \frac { Z } { \sqrt { Y } } \right ] \\ & = \mathbb { E } \left [ Y \frac { Z } { \sqrt { Y } } \right ] - \mathbb { E } [ Y ] \mathbb { E } \left [ \frac { 1 } { \sqrt { Y } } \right ] \mathbb { E } \left [ Z \right ] \\ & = \mathbb { E } [ \sqrt { Y } ] \mathbb { E } [ Z ] \\ & = 0$$

## 929 Correlation Variance

Company : Five Rings

## Prompt:

Suppose we have that σ X = 4 , σ Y = 7 , and σ X + Y = 11 . What is ρ ( X,Y ) ?

## Explanation:

We are going to generalize this to when σ X = a, σ Y = b, and σ X + Y = a + b . We know that

$$V a r ( X + Y ) = V a r ( X ) + V a r ( Y ) + 2 C o v ( X , Y ) \, \Longrightarrow \, C o v ( X , Y ) = \frac { 1 } { 2 } \, ( ( a + b ) ^ { 2 } - a ^ { 2 } - b ^ { 2 } ) = a b$$

After, we know that

$$\rho ( X , Y ) = \frac { C o v ( X , Y ) } { \sigma _ { X } \sigma _ { Y } } = \frac { a b } { a b } = 1$$
