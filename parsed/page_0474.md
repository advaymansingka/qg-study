## 800 Picky Primes

Company : Jane Street, SIG

## Prompt:

4 distinct integers are drawn from the set of the first 16 positive prime integers. Find the probability that the sum is even.

## Explanation:

All primes besides 2 are odd, so we get a sum that is odd precisely when we select integers that are all not 2 . In other words, we select our 4 integers from the other 15 primes. There are ( 15 4 ) ways to pick the 4 from the other 15 and ( 16 4 ) total ways to pick 4 primes from the 16 . Therefore, our probability is

$$\frac { \binom { 1 5 } { 4 } } { \binom { 1 6 } { 4 } } = \frac { 3 } { 4 }$$
