## 221 Poisson Review I

## Prompt:

On average, 7 customers arrive at the QuantGuide gift shop per hour following a Poisson process. To the nearest thousandth, what is the probability that at least 4 customers arrive in a given hour?

## Explanation:

Let X ∼ Poisson (7) . We wish to compute P ( X ≥ 4) = 1 -P ( X ≤ 3) . Recall that the pmf of a Poisson random variable is P [ X = x ] = λ x x ! e -λ .

$$1 - \mathbb { P } ( X \leq 3 ) = 1 - \sum _ { x = 0 } ^ { 3 } \frac { \lambda ^ { x } } { x ! } e ^ { - \lambda } \\$$

≈ 0 . 918

## 222 Cube Pack

Company : Peak6

## Prompt:

How many 3 × 3 × 3 cubes can be fit inside a cube of dimensions 20 × 20 × 20 ?

## Explanation:

In each dimension, we can only fit 6 of the 3 × 3 × 3 cubes. This is because a 7 th would give us a dimension of length 21 &gt; 20 , which is not contained inside the cube. Therefore, the answer is just 6 3 = 216 .
