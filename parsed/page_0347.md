## Prompt:

You repeatedly draw marbles from a bag containing 50 red and 50 blue marbles until there are no more marbles left, recording the order of red and blue marbles drawn. You then count the number of "runs,' where a run is defined as any number of consecutive marbles of the same color. For example, RBBRRRBRR contains 5 runs. What is the expected number of runs that you observe?

## Explanation:

Let us define random variables X 1 , . . . , X 100 such that

$$X _ { i } = \begin{cases} 1 & \text {if the $i$-th draw is the beginning of a new run} \\ 0 & \text {otherwise} \end{cases} \\ \intertext { f R B B R R R R R R , $ for example, $ we would have $ X _ { 1 } = 1 , X _ { 2 } = 1 , X _ { 4 } = 1 , X _ { 7 } = 1 $ }$$

$$Z = \sum _ { i = 1 } ^ { 1 0 0 } X _ { i }$$

In the case of RBBRRRBRR , for example, we would have X 1 = 1 , X 2 = 1 , X 4 = 1 , X 7 = 1 , X 8 = 1 . Note that, for a sequence of 100 red and blue marbles drawn in any order, it is always the case that X 1 = 1 . Let Z represent the total number of runs.

By the linearity of expectation,

$$i = 1 \\ \mathbb { E } [ Z ] & = \mathbb { E } \left [ \sum _ { i = 1 } ^ { 1 0 0 } X _ { i } \right ] \\ & = \sum _ { i = 1 } ^ { 1 0 0 } \mathbb { E } \left [ X _ { i } \right ] \\ & = 1 + \sum _ { i = 2 } ^ { 1 0 0 } \mathbb { E } \left [ X _ { i } \right ]$$

Now let's consider E [ X i ] = P ( X i = 1) for i ≥ 2 .

$$\mathbb { P } ( X _ { i } = 1 ) & = \mathbb { P } ( ( ( i - 1 ) - \text {th draw } R , i - \text {th draw } B ) \cup ( ( i - 1 ) - \text {th draw } B , i - \text {th draw } R ) \\ & = 2 \cdot \mathbb { P } ( ( i - 1 ) - \text {th draw } R , i - \text {th draw } B ) \\ & = 2 \cdot \mathbb { P } ( i - \text {th draw } B | ( i - 1 ) - \text {th draw } R ) \cdot \mathbb { P } ( ( i - 1 ) - \text {th draw } R )$$

The probability that the ( i -1) -th draw is red is simply 50 100 . The probability that the i -th draw is blue given that the previous draw is red is simply 50 99 . Plugging this in, we find

$$\mathbb { P } ( X _ { i } = 1 ) & = 2 \cdot \frac { 5 0 } { 1 0 0 } \cdot \frac { 5 0 } { 9 9 } \\ & = \frac { 5 0 } { 9 9 } \\ & = \mathbb { E } [ X _ { i } ]$$
