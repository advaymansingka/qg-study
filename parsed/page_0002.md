Let I i ( i = 1 , 2 , ..., 5 ) be the indicator random variable where:

$$I _ { i } = \left \{ \begin{array} { l l } { 1 } & { i f the i - t h o t y t o p e x i s t h e s t o v e n b o x e s } \\ { 0 } & { o t h e r w i s e } \end{array}$$
