## Prompt:

How many values of n are there such that a regular n -gon has interior angles that (in degrees) are integervalued?

## Explanation:

The prime factorization of 360 is 360 = 2 3 · 3 2 · 5 . Therefore, there are (3 + 1)(2 + 1)(1 + 1) = 24 total factors of 360 . However, we need to remove 1 and 2 as factors, as those are not polygons. Therefore, there are 24 -2 = 22 values of n that are valid.

## Prompt:

Consider an underlying with S 0 = 25 . We have the following derivatives contract: this contract pays 1 if S T ≥ 24 and -1 otherwise. We have access to the following binary calls (a contract paying 1 if S T ≥ K and 0 otherwise). The calls are given in the format of ( Strike K, Price C 0 )

(25 , 0 . 6)

(24 , 0 . 73)

(23 , 0 . 88)

Give the time0 price of the derivatives contract. Bonds pay 1 at timeT and have time0 price B 0 = 0 . 9 .

## Explanation:

We can create a replication of this. The region where S T ≥ 24 is simple as it is just the binary call. We can imagine the negative region as a binary put. We know that a P 0 + C 0 = B 0 . So, we can price the binary put as P 0 = B 0 -C 0 = 0 . 9 -0 . 73 = 0 . 17 . Since this portion of the payoff is negative, we are shorting the binary put. Combining all the values, we get:

$$V _ { 0 } = C _ { 0 } - P _ { 0 } = 0 . 7 3 - 0 . 1 7 = 0 . 5 6$$

Note: this derivatives contract almost acts like a binary call, but we may have to pay if it expires OTM. This type of contract may be useful when we are looking for a cheaper contract ( 0 . 73 vs. 0 . 56 ).
