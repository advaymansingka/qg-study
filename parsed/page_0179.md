## Prompt:

A casino offers you a game with a six-sided die where you are paid the value of the roll. If you roll a 1, 2, or 3, the game stops and you are paid out. Else, you add the value to your total sum and get to continue rolling. What is the fair value of this game?

## Explanation:

Your payoff is dependent on the first roll, so we can use the Law of Total Expectation. Let x be the fair value of this game. There is a 1 2 probability that you roll a 1, 2, or 3 (on average 2). The other 1 2 of the time, you roll a 4, 5, or 6 (on average 5), add this to your total, and essentially restart the game with an expected value of x . Thus, we can write:

$$x = \frac { 1 } { 2 } \times ( 2 ) + \frac { 1 } { 2 } \times ( x + 5 ) \\ x = 7$$
