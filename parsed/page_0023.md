## Explanation:

The key idea here is that the correlation matrix for any collection of random variables must be positive semi-definite. Denoting Corr ( X,Z ) = ρ , the correlation matrix for ( X,Y,Z ) , say C , is a 3 × 3 matrix

$$\begin{bmatrix} 1 & \frac { 5 } { 1 3 } & \rho \\ \frac { 5 } { 1 3 } & 1 & \frac { 1 2 } { 1 3 } \\ \rho & \frac { 1 2 } { 1 3 } & 1 \end{bmatrix}$$

A condition to show that a matrix is positive definite is that each of the sub-matrices of order 1 × 1 , 2 × 2 , . . . , n × n , where n is a size of the matrix, originating from the top left corner have non-negative determinant. In this case, it is easy to see that the top left 2 × 2 matrix has determinant C 11 C 22 -C 12 C 21 = 144 169 &gt; 0 , so we only need to check that the entire matrix C has non-negative determinant.
