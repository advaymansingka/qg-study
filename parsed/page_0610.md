The surfaces of a 3 × 3 × 3 cube (initially white) are painted red and then is cut up into 27 1 × 1 × 1 small cubes. One of the cubes is selected uniformly at random and is rolled. The face appearing is red. What is the probability the cube selected was a corner cube?

## Explanation:

The key here is to condition on U . This is because of the fact that N | U = u ∼ Geom (1 -u ) , as each trial has probability 1 -u of being larger than u . Therefore, E [ N | U ] = 1 1 -U by known formulas about geometric random variables. Using the Law of Total Expectation and LOTUS, we have that

$$\mathbb { E } [ N ] = \mathbb { E } [ \mathbb { E } [ N \, | \, U ] ] = \mathbb { E } \left [ \frac { 1 } { 1 - U } \right ] = \int _ { 0 } ^ { 1 } \frac { d u } { 1 - u } = \infty$$
