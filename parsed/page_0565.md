## 958 Coin Streak

Company : None

## Prompt:

Find the expected number of flips of a fair coin needed to either have 10 more heads flipped than tails or 7 more tails flipped than heads.

## Explanation:

This is a gambler's ruin problem in disguise. If you obtain $1 for each head and lose $1 for each tail that appears, this is asking the expected time for you to either have $10 or -$7 . The expected hitting time of the set { -a, b } for a, b &gt; 0 is a commonly known to be -ab . In this case, a = 7 and b = 10 , so our expected hitting time is 70 .
