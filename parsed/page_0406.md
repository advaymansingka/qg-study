He runs a statistical test with α = 0 . 05 to test the null hypothesis H 0 : µ = 15 against his alternative hypothesis H a : µ = 16 . What is the probability of a Type II error? $$P ( \bar { x } \leq 1 5 . 8 2 \, | \, H _ { 0 } = 1 6 ) = P ( z \leq \frac { 1 5 . 8 2 - 1 6 } { 3 / \sqrt { 3 6 } } ) = P ( z \leq - 0 . 3 6 ) \approx 0 . 3 5 9 4$$
