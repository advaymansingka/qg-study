## Prompt:

Let W t be a standard Brownian Motion. Suppose that X t is some process such that 1 satisfies the SDE

$$& \text {Let } W _ { t } \text { be a standard Brownian Motion.  Suppose that } X _ { t } \text { is some process such that } \frac { X _ { t } } { X _ { t } } \text { satisfies the SDE} \\ & \quad d \left ( \frac { 1 } { X _ { t } } \right ) = \frac { 1 } { X _ { t } } \left ( 2 d t - d W _ { t } \right ) . \text { The SDE that } X _ { t } \text { satisfies is in the form } d X _ { t } = X _ { t } ( a d t + b d W _ { t } ) \text { for some integers} \\ & \quad a \text { and } b \text { . Find ab.}$$

## Explanation:

$$& \quad \text {Let } Y _ { t } = \frac { 1 } { X _ { t } } . \text { We can write the SDE in the question as } d Y _ { t } = Y _ { t } ( 2 d t - d W _ { t } ) . \text { Note that } X _ { t } = \frac { 1 } { Y _ { t } } \text { from} \\ & \quad \text {definition. Therefore, let } f ( y ) = \frac { 1 } { y } . \text { We have that } f ^ { \prime } ( y ) = - \frac { 1 } { y ^ { 2 } } \text { and } f ^ { \prime \prime } ( y ) = \frac { 2 } { y ^ { 3 } } .$$

From Ito's Formula, we get that dX t = df ( Y t ) = f ′ ( Y t ) dY t + 1 2 f ′′ ( Y t ) d [ Y, Y ] t , where d [ Y, Y ] t is the quadratic variation of Y t . We already know dY t from the SDE above. We get that

$$d [ Y , Y ] _ { t } = ( Y _ { t } ( 2 d t - d W _ { t } ) ) ^ { 2 } = Y _ { t } ^ { 2 } \left ( 4 d [ t , t ] _ { t } - 4 d [ t , W ] _ { t } + d [ W , W ] _ { t } \right ) = Y _ { t } ^ { 2 } d t$$

by the fact that d [ t, t ] t = d [ t, W ] t = 0 .

Plugging everything in,

$$d X _ { t } = - \frac { 1 } { Y _ { t } ^ { 2 } } \cdot Y _ { t } ( 2 d t - d W _ { t } ) + \frac { 1 } { Y _ { t } ^ { 3 } } \cdot Y _ { t } ^ { 2 } d t = \frac { 1 } { Y _ { t } } \left ( - d t + d W _ { t } \right ) = X _ { t } ( - d t + d W _ { t } )$$

This yields that our answer is -1 · 1 = -1 .
