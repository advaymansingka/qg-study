## Prompt:

You flip a fair coin n times. Let X denote the number of heads, and let Y denote the number of tails. E [ XY ] can be expressed in the form

$$\frac { a n ^ { 2 } + b n + c } { d }$$

for some integers a, b, c, d , where a, b, c are each relatively prime with d . What is a + b + c + d ?

## Explanation:

If X heads are flipped out of a total of n coin tosses, then Y = n -X tails are flipped. Let's substitute this expression in for Y , simplify, and apply the linearity of expectation.

$$\mathbb { E } [ X Y ] & = \mathbb { E } [ X ( n - X ) ] \\ & = n \cdot \mathbb { E } [ X ] - \mathbb { E } [ X ^ { 2 } ]$$

Note that X ∼ Binom ( n, 0 . 5) . Recall that for some Z ∼ Binom ( n, p ) , E [ Z ] = np and Var ( Z ) = npq , where q = 1 -p . So,

We can compute E [ X 2 ] as follows:

$$\mathbb { E } [ X ] & = \frac { n } { 2 } \\ \text {Var} [ X ] & = \frac { n } { 4 }$$

$$Var [ X ] & = \mathbb { E } [ X ^ { 2 } ] - ( \mathbb { E } [ X ] ) ^ { 2 } \\ & = \frac { n } { 4 } \\ \Rightarrow \mathbb { E } [ X ^ { 2 } ] & = \frac { n } { 4 } + \left ( \frac { n } { 2 } \right ) ^ { 2 } \\ \intertext { o n f i n e s t , }$$

Returning to our original expression of interest,

$$\mathbb { E } [ X Y ] & = \mathbb { E } [ X ( n - X ) ] \\ & = n \cdot \mathbb { E } [ X ] - \mathbb { E } [ X ^ { 2 } ] \\ & = \frac { n ^ { 2 } } { 2 } - \frac { n ^ { 2 } } { 4 } - \frac { n } { 4 } \\ & = \frac { n ^ { 2 } - n } { 4 } .$$

Our solution is therefore 1 -1 + 4 = 4 .
