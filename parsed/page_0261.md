The expected gain can then be computed via the law of total expectation.

$$\mathbb { E } [ G ] = \mathbb { E } [ G | X > 0 . 5 ] \mathbb { P } ( X > 0 . 5 ) + \mathbb { E } [ G | X \leq 0 . 5 ] \mathbb { P } ( X \leq 0 . 5 )$$
