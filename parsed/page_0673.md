## 1139 Cheese Lover III

Company : DRW, SIG, AQR

## Prompt:

Jon loves cheese. He decides to make 100 blocks of cheese. The distribution of the weight (in grams) of each block he makes follows IID Exp ( 1 250 ) distribution. Let W i denote the weight of the i th block of cheese, and T 100 represent the total weight of the 100 blocks of cheese. Using Central Limit Theorem, what is an approximation for P [ T 100 &gt; 26000] ? The answer is in the form Φ ( a ) for some real a . Φ here is the CDF of the standard normal distribution. Find a .

## Explanation:

We subtract the expectation of T 100 = 25000 from both sides, then divide by Var ( T 100 ) = 100(250) 2 (by recognition of using linearity of variance), implying that σ T 100 = √ 100(250) 2 = 250 √ 100 = 2500 . Thus. we get that

$$\mathbb { P } [ T _ { 1 0 0 } > 2 6 0 0 ] = 1 - \mathbb { P } [ T \leq 2 6 0 0 ] = 1 - \mathbb { P } [ T _ { 1 0 0 } - 2 5 0 0 \leq 2 6 0 0 - 2 5 0 0 ] = 1 - \mathbb { P } \left [ \frac { T _ { 1 0 0 } - 2 5 0 0 } { 2 5 0 0 } \leq \frac { 1 0 0 0 } { 2 5 0 0 } \right ]$$

We approximate the LHS using a standard normal random variable by the CLT, so we are using the approximation 1 -P [ Z ≤ 2 5 ] , where Z ∼ N (0 , 1) . This last quantity is 1 -Φ ( 2 5 ) = Φ ( -2 5 ) . Therefore, a = -2 / 5 .
