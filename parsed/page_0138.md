## Explanation:

Let random variable X denote the minimum number rolled after you repeatedly rolls a die until you get a 5 . Notice that P ( X = 6) = 0 , since a 5 will always be rolled. By the definition of expectation, we have:

$$\mathbb { E } [ X ] = \sum _ { x = 1 } ^ { 5 } x \cdot \mathbb { P } ( X = x ) .$$

Let's compute P ( X = 1) . In other words, we want to find the probability that there exists at least one 1 before a 5 occurs. Notice that there are only two possibilities (we don't care what the numbers between 1 and 5 actually are): (1) a 1 appears before a 5 , or (2) a 5 appears before a 1 . By symmetry and the law of total probability, we conclude that P ( X = 1) = 1 / 2 .

Next, let's compute P ( X = 2) . In order for 2 to be the minimum value, a 1 cannot be rolled before the first 5 , and a 2 must be rolled before the first 5 . Consider an arbitrary sequence of rolled values of infinite length that contains at least one each of 1 , 2 , and 5 . There are a couple possibilities. Just to name a few: (1) 1 appears before 2 which appears before 5 , and (2) 2 appears before 5 which appears before 1 , and (3) 2 appears before 1 which appears before 5 . In total, there are 3! = 6 ways to order { 1 , 2 , 5 } . Only when we have { 2 , 5 , 1 } will we achieve X = 2 . By symmetry and the law of total probability, for an arbitrary sequence of infinite length, the probability that a 2 appears before a 5 which appears before a 1 is 1 / 6 .

Similarly, for P ( X = 3) , we need to find orderings of { 1 , 2 , 3 , 5 } such that 3 appears before 5 which appears before 1 and 2 in any order. Therefore, P ( X = 3) = 2! / 4! = 1 / 12 . Repeating this logic for P ( X = 4) and P ( X = 5) , we find P ( X = 4) = 3! / 5! = 1 / 20 and P ( X = 5) = 4! / 5! = 1 / 5 . Putting it all together, we find:

$$\mathbb { E } [ X ] & = \sum _ { x = 1 } ^ { 5 } x \cdot \mathbb { P } ( X = x ) \\ & = \frac { 1 } { 2 } + 2 \cdot \frac { 1 } { 6 } + 3 \cdot \frac { 1 } { 1 2 } + 4 \cdot \frac { 1 } { 2 0 } + 5 \cdot \frac { 1 } { 5 } \\ & = \left [ \frac { 1 3 7 } { 6 0 } \right ] .$$
