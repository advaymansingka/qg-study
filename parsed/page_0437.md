Our solution is therefore

$$Var ( 2 X _ { 1 } - 4 X _ { 2 } ) & = 4 \text {Var} ( X _ { 1 } ) + 1 6 \text {Var} ( X _ { 2 } ) - 1 6 \text {Cov} ( X _ { 1 } , X _ { 2 } ) \\ & = \frac { 1 2 } { 8 0 } + \frac { 1 6 } { 2 0 } - \frac { 1 6 } { 4 0 } \\ & = \frac { 1 1 } { 2 0 } .$$
