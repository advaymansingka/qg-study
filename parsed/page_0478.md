## Prompt:

A rope that is one meter long is divided into three segments by two random points. What is the expected length of the longest segment?

## Explanation:

Let X ∈ [0 , 1] be the location of the first cut and Y ∈ [0 , 1] be the location of the second cut such that X &lt; Y . Because each segment is equally likely to be the longest, the expected length of the longest piece doesn't depend on which piece we choose, so we can solve for E [ X | X is the longest ] . This, with our earlier constraint of X &lt; Y , gives us two additional constraints:

$$X & > Y - X \Rightarrow Y < 2 X \\ X & > 1 - Y \Rightarrow Y > 1 - X$$

These two constraints derive from the fact that X is given to be greater than the other two segments, Y -X and 1 -Y . Graphing the constraints within our sample space, we see that the area satisfying our inequalities is the triangle A with vertices ( 1 2 , 1 2 ) , ( 1 2 , 1) , and ( 1 3 , 2 3 ) , and triangle B with vertices ( 1 2 , 1 2 ) , ( 1 2 , 1) , and (1 , 1) . We wish to find the probability of an outcome occurring in either areas and the expected value of X within each area.

$$\ A r e a \, o f \, A \colon \frac { 1 } { 2 } \times \frac { 1 } { \frac { 1 } { 2 } } \times ( \frac { 1 } { \frac { 1 } { 2 } } - \frac { 1 } { \frac { 1 } { 3 } } ) = \frac { 1 } { 2 4 }$$

$$Area of A \colon \frac { 1 } { 2 } \times \frac { 1 } { 2 } \times ( \frac { 1 } { 2 } - \frac { 1 } { 3 } ) = \frac { 1 } { 2 4 } \\ \intertext { A r e a f $ B \colon \frac { 1 } { 2 } \times \frac { 1 } { 2 } \times \frac { 1 } { 2 } = \frac { 1 } { 8 } = \frac { 3 } { 2 4 } = 3 \times A r e a f $ A } \\ \intertext { E x p e c t e d v u l e f $ X $ with h i n $ A \colon \frac { 1 } { 2 } - \frac { 1 } { 3 } ( \frac { 1 } { 2 } - \frac { 1 } { 3 } ) = \frac { 4 } { 9 } } \\ \intertext { E x p e c t e d v u l e f $ X $ with h i n $ B \colon \frac { 1 } { 2 } + \frac { 1 } { 3 } \times \frac { 1 } { 2 } = \frac { 2 } { 3 } } \\ \intertext { U l e f $ X $ w i n h i n e a t h i n g l e s a l c u l l a s u c t }$$

The expected value of X within each triangle is calculated using the property of centroids. A centroid, or the center of area for a triangle, is 1 3 of the way from the base to the opposite vertex, a property that is most easily seen by integration. Finally, we can solve for the expected value of X given X is the longest piece:

$$E [ X | X \text { is the longest} ] = \frac { 1 } { 4 } \times \frac { 4 } { 9 } + \frac { 3 } { 4 } \times \frac { 2 } { 3 } = \frac { 1 1 } { 1 8 }$$
