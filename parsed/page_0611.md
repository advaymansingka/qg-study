## 1038 Product and Sum

Company : Belvedere Trading

## Prompt:

Suppose x and y satisfy x + y = 10 and xy = 20 . Find the value of x 4 + y 4 .

## Explanation:

Note that

$$( x + y ) ^ { 4 } = x ^ { 4 } + 4 x ^ { 3 } y + 6 x ^ { 2 } y ^ { 2 } + 4 x y ^ { 3 } + y ^ { 4 } = ( x ^ { 4 } + y ^ { 4 } ) + 2 x y \left [ 2 x ^ { 2 } + 3 x y + 2 y ^ { 2 } \right ] = ( x ^ { 4 } + y ^ { 4 } ) + 2 x y \left [ 2 ( x + y ) ^ { 2 } - x y \right ] \\ \\ \text {Since} \, \ m { \log } \, \ t h { x } \, \ m { \log } \, \ t h { y } \, \ m { \log } \, \ t h { x } \, \ t h { y } ^ { 4 } + 4 y ^ { 4 } - ( \ln \, y ) ^ { 4 } \, \ m { \log } \, \ t h { y } \, \ t h { x } \, + 2 x y \, \ t h { y } ^ { 2 } + 4 y ^ { 2 } \, - ( \ln \, y ) ^ { 4 } \, \ m { \log } \, \ t h { y } ^ { 2 } \, + y ^ { 2 } \right ]$$

Since we know the values of x + y and xy , we can say that x 4 + y 4 = ( x + y ) 4 -2 xy [ 2( x + y ) 2 -xy ] and substitute in the values. Namely, we get that x 4 + y 4 = 10 4 -2(20)(2 · 100 -20) = 10000 -7200 = 2800

## 1039 Better in Red II

Company : SIG, Jane Street, Hudson River Trading, Five Rings

## Prompt:

A 10 × 20 × 30 rectangular prism is painted red on the surface and then cut into 6000 1 × 1 × 1 cubes and one is selected uniformly at random. Find the expected number of red faces on this cube.

## Explanation:

Label the faces of each cube 1 -6 , and then let I i be the indicator that side i of the cube that is drawn is colored red. Then T = I 1 + · · · + I 6 gives the total number of red sides of the cube. We need to be careful here, as the indicators are not exchangeable. Instead, two of the indicators will correspond to a 10 × 20 side, two to a 20 × 30 side, and two to a 10 × 30 side. Therefore, there are 3 subsets of indicators that are exchangeable, so we can reduce it to an expectation involving 3 indicators and multiply it by 2, so E [ T ] = 2( E [ I 1 ] + E [ I 2 ] + E [ I 3 ]) , where we take I 1 to indicate a 10 × 20 side, I 2 to indicate a 20 × 30 side, and I 3 to indicate a 10 × 30 side.

Then, to evaluate these expectations, we just need to find the probability that the side indicated is red on our cube. For I 1 , that is 10 · 20 = 200 cubes. For I 2 , that is 20 · 30 = 600 cubes. For I 3 , that is 10 · 30 = 300 cubes. All of these need to be divided by 6000 as that is the volume of the entire prism. Therefore, E [ T ] = 2(200 + 300 + 600) 6000 = 11 30 .
