## Prompt:

Find the derivative of f ( x ) = (ln( x )) x at x = e .

## Explanation:

The trick here is to use logarithmic differentiation. In other words, let y = f ( x ) = (ln( x )) x . Then ln( y ) = x ln(ln( x )) . Using the product and chain rules, we have that

$$\frac { y ^ { \prime } } { y } = \ln ( \ln ( x ) ) + x \cdot \frac { 1 } { x \ln ( x ) } \iff y ^ { \prime } = y \left [ \ln ( \ln ( x ) ) + \frac { 1 } { \ln ( x ) } \right ] = ( \ln ( x ) ) ^ { x } \left [ \ln ( \ln ( x ) ) + \frac { 1 } { \ln ( x ) } \right ]$$

Plugging in x = e , the above expression evaluates to 1 .

## Prompt:

Let's say you have product A in which you are quoting 4 @ 5 and product B in which you are quoting 10 @ 12 . X @ Y means that X is our bid and Y is our ask. We want to make a market on the product A + B . What is the bid-ask spread you will quote? Give the answer in the format of Y 2 -X 2

## Explanation:

We want no arbitrage pricing, so our pricing should be the same as a replicated portfolio. In other words, the price of Price A + B = Price A + Price B . There are two perspectives we can take: a market-maker and a market-taking perspective. The market-maker perspective is more important in the context of trading interviews, so we will focus on this.

When we consider A + B , we are considering a long position of A and a long position of B . So, if we want to set our bid (and not be taken out by other market-makers), we should set our bid to be the Bid A + Bid B and ask to be Ask A + Ask B . This gives us an ask of 17 and a bid of 14 and so Y 2 -X 2 = 17 2 -14 2 = 93 .
