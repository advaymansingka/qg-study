## 39 Absolute Difference

Company : DRW

## Prompt:

Let x 0 = 0 and let x 1 , . . . , x 10 satisfy that | x i -x i -1 | = 1 for 1 ≤ i ≤ 10 and x 10 = 4 . How many such sequences are there satisfying these conditions?

## Explanation:

Let's put this problem in a different light. Consider a walk where you move up 1 for heads and move down 1 for tails. The condition states that in the 10 fl ips, we have 4 more heads than tails, so this means that there are 7 movements up and 3 movements down. In other words, each valid sequence corresponds to some ordering of UUUUUUUDDD . There are ( 10 3 ) = 120 such arrangements, so this is our answer.

## 40 Counting Digits

## Prompt:

How many digits are in 99 to the 99th power?

## Explanation:

There are 198 digits in 99 99 .

$$9 9 ^ { 9 9 } = 9 9 ^ { 9 9 } \times ( \frac { 1 0 0 } { 1 0 0 } ) ^ { 9 9 } = 1 0 0 ^ { 9 9 } \times ( \frac { 9 9 } { 1 0 0 } ) ^ { 9 9 } = 1 0 ^ { 1 9 8 } \times ( 1 - \frac { 1 } { 1 0 0 } ) ^ { 9 9 } \approx 1 0 ^ { 1 9 8 } \times ( 1 - \frac { 1 } { 1 0 0 } ) ^ { 1 0 0 } \approx 1 0 ^ { 1 9 8 } \times \frac { 1 } { e }$$

Thus, 99 99 has 198 digits.
