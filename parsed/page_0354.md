## Prompt:

Consider a uniformly random permutation of (1 , 2 , . . . , 1000) . Let X be the number of fixed points and Y be the number of non-fixed points. Compute Var ( X -Y ) .

## Explanation:

Note that Y = 1000 -X , as X + Y = 1000 always. Therefore, Var ( X -Y ) = Var (1000 + 2 X ) = 4 Var ( X ) . It now remains to compute Var ( X ) . Note that X = X 1 + · · · + X 1000 , where X i is the indicator of whether or not i is fixed. Then E [ X ] = 1000 E [ X 1 ] by exchangeability. E [ X 1 ] is just the probability 1 is fixed, which is 1 1000 , as it is equally likely to end in any of the 1000 spots.

To compute E [ X 2 ] , we expand out the sum squared. Namely,

$$\mathbb { E } [ X ^ { 2 } ] = \mathbb { E } [ ( X _ { 1 } + \cdots + X _ { 1 0 0 } ) ^ { 2 } ] = \sum _ { i = 1 } ^ { 1 0 0 } \mathbb { E } [ X _ { i } ^ { 2 } ] + \sum _ { i \neq j } \mathbb { E } [ X _ { i } X _ { j } ]$$

The first term is where i = j and the second term is where i = j . Note that X 2 i = X i , as X i only takes the values 0 and 1 , so E [ X 2 i ] = 1 1000 .

This probability is just 1 1000 · 1 999 , as once 1 is fixed, 2 goes into spot 2 with probability 1 999 . Therefore, E [ X 1 X 2 ] = 1 1000 · 1 999 .
