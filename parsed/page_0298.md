On the denominator, we need to condition on whether or not we receive the double-headed coin. Thus,

$$\mathbb { P } [ H _ { n } ] = \mathbb { P } [ H _ { n } \ | \ D ] \mathbb { P } [ D ] + \mathbb { P } [ H _ { n } \ | \ D ^ { c } ] \mathbb { P } [ D ^ { c } ]$$

We already calculated the first term in the numerator. For the second term, we know P [ D c ] = 1 -P [ D ] = 19 20 . Then, P [ H n | D c ] = 1 2 n , as we flip a heads on a fair coin with probability 1 2 in each turn. Therefore, P [ H n ] = 1 20 + 19 20 · 2 n . Therefore, p n = 1 20 1 20 + 19 20 · 2 n = 1 1 + 19 2 n . Our goal is to find the smallest n such that p n &gt; 19 20 .

This means that 1 1 + 19 2 n &gt; 19 20 . Multiplying by the denominator on both sides yields 1 &gt; 19 20 + 361 20 · 2 n . Multiplying by 20 · 2 n on both sides yields 2 n &gt; 361 , which occurs for n ≥ 9 . This means 9 times is the minimum number we need.
