Now, by sub-additivity and the fact that the collection of all infinite components passing through the origin is a subset of the length n components passing through the origin (and then the rest of the path is irrelevant after that),

$$\mathbb { P } [ A ] \leq \mathbb { P } \left [ \bigcup _ { \gamma } A _ { \gamma } \right ] \leq \sum _ { \gamma } \mathbb { P } [ A _ { \gamma } ] = \sum _ { \gamma } p ^ { n } \\$$

How many paths γ are there of length n passing through the origin? An easy upper bound is to note that the initial direction has 4 choices, and then after that, there are at most 3 other directions that the path can go at each step, as it can't go backwards to the spot it was before. Therefore, an upper bound on the number of such paths is 4 · 3 n -1 . Therefore P [ A ] ≤ 4 · 3 n -1 · p n = 4 3 · (3 p ) n . For this probability to tend to 0 , we need 3 p &lt; 1 , so p &lt; 1 3 . Therefore, our interval is ( 0 , 1 3 ) , so a = 1 3 .
