## Prompt:

Let W t be a standard Brownian Motion. We define the Brownian Bridge on [0 , 1] as X t = W t -tW 1 . Find E [ X 1 / 2 | X 3 / 4 = 3] .

## Explanation:

We will solve this more generally for 0 &lt; s &lt; t &lt; 1 . Write X s = W s -sW 1 = ( W s -s t W t ) + s t ( W t -tW 1 ) = ( W s -s t W 1 ) + s t X t . For fixed times, both of the terms are normally distributed. Furthermore,

$$C o v \left ( W _ { s } - \frac { s } { t } W _ { t } , X _ { t } \right ) = C o v \left ( W _ { s } - \frac { s } { t } W _ { t } , W _ { t } - t W _ { 1 } \right ) = C o v ( W _ { s } , W _ { t } ) - t C o v ( W _ { s } , W _ { 1 } ) - \frac { s } { t } C o v ( W _ { t } , W _ { t } ) + s C o v ( W _ { t } , W _ { 1 } )$$

$$= s - s t - s + s t = 0$$

Therefore, we have that W s -s t W t and X t are independent, as they are uncorrelated normals. Thus, E [ W s -s t W t | X t ] = E [ W s ] -s t E [ W t ] = 0 . Using this, we can find that

$$\mathbb { E } [ X _ { s } | \ X _ { t } ] = \mathbb { E } \left [ \left ( W _ { s } - \frac { s } { t } W _ { 1 } \right ) + \frac { s } { t } X _ { t } \, | \, X _ { t } \right ] = \frac { s } { t } X _ { t } \\$$

We obtain this from the fact that the first term vanishes by the above and that X t is known in the second term. In particular, we have that s = 1 / 2 , t = 3 / 4 , and X t = 3 , so our answer is 2 3 · 3 = 2 .
