## Prompt:

Let X denote an asset that has the following dynamics with X 0 = 2 .

$$d X _ { t } = 5 d t + 2 d W _ { t }$$

where dW t is a standard Brownian motion.

Find the mean µ T and variance σ 2 T of X T , where T = 0 . 2 . Give the answer in the format of µ T ∗ σ 2 T .

## Explanation:

When we have an asset following dynamics dX t = a dt + b dW t , where a, b ∈ R , we have the following properties.

$$\mu _ { T } & = X _ { 0 } + a T \\ \sigma _ { T } ^ { 2 } & = b ^ { 2 } T$$

We can also obtain these results intuitively without using the fact that a standard Brownian Motion follows W t ∼ N (0 , t ) . The Brownian motion component is the randomness contributing part, with mean 0 . Thus, we can expect our time T value to increase in direction by a dt with an initial position of X 0 . We can then use properties of a normal distribution to obtain the variance of our time T position (as the dW t term contributes solely to the randomness of our time T position. We can see that b N (0 , t ) = N (0 , b 2 t ) .

## Prompt:

Suppose N &gt; 0 is an integer. Let X 1 , X 2 , X 3 be independently selected uniformly at random from the set { 0 , 1 , . . . , N } . Let p N = P [ X 1 + X 2 + X 3 = N ] . Find lim N →∞ Np N . If this limit does not exist, enter -1 .

## Explanation:

For X 1 + X 2 + X 3 = N to occur, we would need X 1 + X 2 ≤ N and X 3 = N -X 1 -X 2 to occur simultaneously. This is because all of the values we select are non-negative. Let A N = { X 1 + X 2 ≤ N } and B N = { X 3 = N -X 1 -X 2 } . Then we want

$$\mathbb { P } [ A _ { N } \cap B _ { N } ] = \mathbb { P } [ B _ { N } \ | \ A _ { N } ] \mathbb { P } [ A _ { N } ]$$

P [ B N | A N ] asks for the probability X 3 = N -X 1 -X 2 given X 1 + X 2 ≤ N . This probability is just 1 N +1 since X 3 is discrete uniform over { 0 , 1 , . . . , N } and the requested value is in the support since we know 0 ≤ X 1 + X 2 ≤ N , meaning N -X 1 -X 2 is too.
