## 169 Crazy Covariance

Company : None

## Prompt:

Let X ∼ Exp (1) and Y | X = x ∼ LogNorm (0 , x ) . Find Cov ( X,Y ) . Note that we say a random variable R ∼ LogNorm ( µ, σ 2 ) if log( R ) ∼ N ( µ, σ 2 ) .

## Explanation:

By definition, we know that Cov ( X,Y ) = E [ XY ] -E [ X ] E [ Y ] . We only know the distribution of Y | X = x . Therefore, we will need to apply LOTUS on any terms involving Y .

Starting with E [ Y ] , E [ Y ] = E [ E [ Y | X ]] . One can show from the definition of the lognormal distribution that for R ∼ LogNorm ( µ, σ 2 ) , E [ R ] = e µ + 1 2 σ 2 . In this case, µ = 0 and σ 2 = X . Therefore, E [ Y | X ] = e X 2 . This implies that E [ Y ] = E [ e X 2 ] . This is just the MGF of X evaluated at θ = 1 2 . As we know X ∼ Exp (1) , the MGF of X is M X ( θ ) = 1 1 -θ . Plugging in θ = 1 2 yields E [ Y ] = 2 . We already know E [ X ] = 1 by the known properties of exponential distributions.

To calculate E [ XY ] , we need to apply LOTUS once again. Thus,

$$\mathbb { E } [ X Y ] = \mathbb { E } [ \mathbb { E } [ X Y \ | \ X ] ] = \mathbb { E } [ X \mathbb { E } [ Y \ | \ X ] ] = \mathbb { E } \left [ X e ^ { \frac { X } { 2 } } \right ]$$

The last expression above is just the first derivative of the MGF of X evaluated at θ = 1 2 . Taking the derivative of the MGF of X , we get M ′ X ( θ ) = 1 (1 -θ ) 2 . Letting θ = 1 2 yields E [ XY ] = 4 . This means Cov ( X,Y ) = 4 -2 · 1 = 2 .
