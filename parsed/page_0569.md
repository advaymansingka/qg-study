## Explanation:

We will offer an informal justification. Recall the definition of conditional probability, where A is an event:

$$\mathbb { E } [ X | A ] = \int _ { x } \mathbb { P } [ X = x | \, A ] \, d x .$$

Let's say that you are told that sin( X ) = 1 / 2 . What can we say about X ? We might be able to use arcsin here.

$$X = \arcsin \left ( \frac { 1 } { 2 } \right ) .$$

However, we also know that there could be multiple possible values of X within the interval [0 , π ] such that sin( X ) = 1 / 2 . Looking at a plot of sin( X ) , we notice that sin( X ) = 1 / 2 when X = arcsin(1 / 2) and when X = π -arcsin(1 / 2) . Replacing 1 / 2 with some general value a ∈ [0 , π ] , we find that there are two values for X such that sin( X ) = a : arcsin( a ) and π -arcsin( a ) . The only exception to this observation is when sin( X ) = 1 . Then, X = π / 2 . So, given sin( X ) = a where a ∈ [0 , 1) , E [ X | sin( X ) = a ] = (arcsin( a )+ π -arcsin( a )) / 2 = π / 2 . And, of course, in the case that we're given sin( X ) = 1 , we have E [ X | sin( X ) = 1] = π / 2 . Therefore, E [ X | sin( X )] = π / 2 , and cos( π 2 ) = 0 .
